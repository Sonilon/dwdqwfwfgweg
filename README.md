# HolyinvisibleMod (Minecraft 1.21.4)

Клиентский мод для **Minecraft Fabric 1.21.4**. Версия для **1.20.1** — в отдельном репозитории/ветке.

Показывает объёмный красный **нимб** (кольцо-тор) над головой невидимых игроков — без силуэта, модели или контура самого игрока. Нимб отрисовывается как обычная 3D-геометрия с проверкой глубины, поэтому он **не виден сквозь стены** и корректно перекрывается блоками мира.

**Автор:** 1pirs

---

## Возможности

- 🕊️ Показывает только сам нимб — тело, руки, броня и прочие части невидимого игрока остаются полностью невидимыми.
- 🔴 Идеально круглый, объёмный красный нимб (настоящая 3D-геометрия тора, а не плоский диск).
- 🧱 Нимб скрывается за блоками — никакого "вижу сквозь стены".
- ✨ Лёгкая анимация: вращение и мягкая пульсация свечения.
- ⌨️ Клавиша быстрого включения/выключения отображения нимбов (по умолчанию **H**), настраивается в **Настройки → Управление → HolyinvisibleMod**.
- 💻 Полностью клиентская логика — не требует установки на сервер, ничего не отправляет по сети.

## Установка

1. Установите [Fabric Loader](https://fabricmc.net/use/) для Minecraft 1.21.4.
2. Установите [Fabric API](https://modrinth.com/mod/fabric-api) той же версии Minecraft.
3. Скачайте `.jar` мода со страницы [Releases](../../releases) или из последнего успешного [Actions-билда](../../actions).
4. Поместите файл в папку `mods` вашей игры.

## Сборка из исходников

Требуется JDK 21+ и [Gradle](https://gradle.org/install/) 8.11+ (или запустите `gradle wrapper` один раз, чтобы сгенерировать `gradlew`/`gradlew.bat` для проекта).

```bash
git clone https://github.com/1pirs/HolyinvisibleMod.git
cd HolyinvisibleMod
gradle wrapper   # один раз, генерирует gradlew — можно закоммитить в репозиторий
./gradlew build
```

Готовый `.jar` появится в `build/libs/`.

Сборка также автоматически запускается через **GitHub Actions** при каждом push в `main` — собранный jar можно скачать во вкладке [Actions](../../actions) как артефакт, а при создании тега `vX.Y.Z` он автоматически прикрепляется к [Release](../../releases).

## Управление

| Действие | Клавиша по умолчанию |
|---|---|
| Включить/выключить показ нимбов | `H` |

Клавишу можно переназначить в **Настройки → Управление → HolyinvisibleMod**.

## Технические детали

- Fabric API `WorldRenderEvents.AFTER_ENTITIES` используется для отрисовки нимба поверх обычной геометрии мира с включённым depth-тестом.
- Нимб рендерится только для игроков, у которых `Entity#isInvisible()` возвращает `true` (эффект невидимости), включая случаи ваниллы и большинства других модов/плагинов, использующих стандартный флаг невидимости.
- Настройка вкл/выкл хранится в памяти клиента и переключается локально, не требует синхронизации с сервером.
